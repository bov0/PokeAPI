Pokedex

Funcionamiento de la página

Se despliegan 8 objetos aleatorios del archivo json, cada uno de ellos se muestra mediante sus datos correspondidos por su ID. Cada uno de ellos tiene un boton el cual si clicas sobre el se despliega otra página dinámica con información adicional del objeto(pokemon).

API y recursos utilizados

API utlizada: PokeAPI

El desarrollo de esta página web fue posible gracias al uso de APIs que nos permiten obtener información detallada sobre los Pokémon, y a recursos como HTML, CSS y JavaScript para diseñar y programar la página.
